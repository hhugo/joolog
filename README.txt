joolog - Logger for js_of_ocaml
===============================

Joolog (Js_of_ocaml logger) is a library managing Lwt logger for application
running in the browser.

See the file [INSTALL.txt](INSTALL.txt) for building and installation
instructions.

[Home page](https://github.com/hhugo/joolog)
